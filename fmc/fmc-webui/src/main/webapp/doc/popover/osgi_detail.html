OSGi Details

Lists the OSGi bundles and OSGi services deployed into the container.

Bundles

Lists all of the OSGi bundles deployed into the container along with the bundle's:

  • ID
  • Version
  • Lifecycle State

Selecting one of the bundles from the list will display details about the bundle on the right of the page. The details include:

  • ID
  • Symbolic Name
  • Version
  • Imported Packages
  • Exported Packages
  • Manifest Headers

Services

Lists all of the services registered with the OSGi service registry. The list is organized by service ID and lists all of the object classes available through the service.

Selecting one of the services from the list will display details about the service on the right of the page. The details include:

  • Service ID
  • ID of the bundle registering the service
  • Object classes available from the service
  • IDs of any bundles used by the service
  • Service properties
