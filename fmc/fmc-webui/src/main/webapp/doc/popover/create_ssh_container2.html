SSH Server Configuration (continued)

Specify where on the remote server the new container will stores its artifacts. Also, specify how hard to try to connect to the remote server.

You need to specify:

  • Path - the folder on the remote server where the data for the root container will be stored.

Important - You must have write and execute privileges for this folder.

  • Number of retries - the number of times Fuse Fabric will attempt to connect to the remote server
  • Retry delay - the number of seconds that Fuse Fabric will wait between attempts to connect to the remote server
