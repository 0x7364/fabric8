Select parent for the child container

Select the parent for the new container from one of the listed roots. The list provides the following details about each root container:

  • Name - the unique identifier of the container
  • Location - a user entered string that should provide information about the location of the container either physically or conceptually
  • Version - the version of the profiles deployed into the container

Note - The version of the child's profiles do not need to match the version of the parent's profiles.
