Users

This page is used to manage the users who can access the Fuse Management Console. Users are all assigned roles that control what they can do while viewing the console. For example, users in the admin role my be able to assign profiles to a container while users in the ituser role may only be able to view details about deployed containers

The page is broken down into three sections:

  • Controls - buttons that allow you to edit the users and roles
  • Users - lists all of the configured users
  • Details - show the details of the selected user

Controls

The controls are located across the top of the page. They consist of four buttons:

  • Create User - Creates a new user
  • Delete User - Deletes the selected user
  • Add Role - Add a new role to the selected user
  • Change Password - Changes the password of the selected user

Users

This table lists all of the console's users. It details:

  • The user's ID
  • The number of roles assigned to the user

Details

This view displays the user name and all of the roles assigned to the selected user.
