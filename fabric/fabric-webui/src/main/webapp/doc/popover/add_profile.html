Add Profiles to Container

Assigns the selected profiles to a container.

Note - Profiles can be deleted from a container on the Fuse Container Instance page or the Container Details page.

A container's profiles are all part of the same version. So, all of the profiles listed in the wizard are from the container's profile version.

To add a profile to a container:

  1. Select one or more profiles from the list.
  2. Click Add.

Tip - If a profile is already deployed into the container, it will not be in the list.
