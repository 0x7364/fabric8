Container Details

Information

This view provides basic details about a container. This information includes:

  • the container's profiles
  • the container's runtime environment
    • PID
    • JVM Version
    • Up time
    • System Architecture
    • System OS
    • Number of available CPU cores
  • the container's name and status
  • the container's system resources
    • physical memory
    • swap space
    • heap memory
    • file descriptors
    • threads

Controls

This page offers the following controls:

  • Add Profiles - Add profiles to the container.
  • OSGi Details - Opens a new page that lists the OSGi bundles and OSGi services deployed into the container
  • Fuse MQ Details - Available if the container has a Fuse MQ Enterprise broker. Opens a new page listing details about the Fuse MQ broker instance.
