Create New User

Creates a new Fuse Management Console user.

You need to provide the following:

  • Username - the unique name the user will use to access the console

Note - The username must be at least five(5) characters long.

  • Password - the password the user will use to access the console

Note - The password must be at least six characters long. It cannot contain spaces.

The new user will not be assigned any roles. You will need to assign the roles using the Add Role button on the User Page.
