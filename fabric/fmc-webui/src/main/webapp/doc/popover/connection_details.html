Connection Details

This page details all of the connectors available to a broker. It displays the information in three tables:

  • Available Connectors - lists the connectors available to the broker by protocol
  • Open Connections - lists the address for all of the connections that are currently open
  • Connection Details - right side table that displays detailed information about the selected conneciton

The right hand table displays detailed information about the selected connection:

  • protocol of connector
  • address from which the client is accessing the connection
  • slow consumer warning - slow consumers can cause a number of problems for a broker
  • blocked transport warning - TCP connections can become blocked by network issues and blocked connections are closed if they are not unblocked in a specified ammount of time
  • dispatch queue size
