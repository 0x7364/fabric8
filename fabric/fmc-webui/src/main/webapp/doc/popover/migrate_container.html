Migrate Fuse Containers

Migrates the selected containers to a new profile version.

Important - If a container has multiple profiles deployed on it, then this will update all of the profiles deployed on the container to the specifications in the new version.

To migrate a container:

  1. Select the container from the list on the left.
  2. Select the new version from the list on the right.
  3. Click Apply.

Cancel returns you to the Fuse Container Instances page.
