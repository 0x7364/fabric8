Create New Version

Creates a new profile version based on a parent version.

All of the profiles from the parent version are copied into the new version to provide a stable, well known base. Once the new version is created, you can edit individual profiles without effecting running containers.

To create a new profile version:

  1. Assign the version a unique name.
  2. Select the version from which the new version will originate.
  3. Click Create.
